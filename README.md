# Offie

**Download YouTube audio and video at the highest quality, straight from your Mac.**

Native SwiftUI app for macOS — fast, self-contained, available in Italian and English.

**[offie.chriszeta.it](https://offie.chriszeta.it)**

---

## Download

Go to [**Releases**](../../releases) to download the latest version.

---

## Features

**Audio** — M4A, M4A Max, OPUS, MP3, MP3 320k, FLAC, WAV. Metadata and cover art embedded automatically.

**Video** — 360p to 4K (2160p), auto-merged to MP4.

**Search** — Search YouTube without an API key, or paste a link directly.

**Queue** — Parallel downloads (1–5), real-time progress with speed and ETA, per-item log.

**Split** — Divide any audio or video file into parts of N minutes, metadata and cover art preserved in every part.

**Premium formats** — Optional: use your Firefox YouTube cookies for high-bitrate Premium streams.

**Self-contained** — Ships with yt-dlp and ffmpeg bundled. No Homebrew required (but supported for extra speed).

**Bilingual** — Italian and English, instant toggle, no restart.

## Requirements

- macOS 14 or later
- Apple Silicon or Intel (universal binary)

---

## Installation

1. Download `Offie.dmg` from [Releases](../../releases)
2. Open the DMG and drag **Offie.app** to your Applications folder
3. **Offie is not notarized by Apple.** On first launch macOS will block it. Open it anyway: go to **System Settings → Privacy & Security → Open Anyway**, or right-click the app → Open → Open
